Enable authentication and security features for the webhook endpoint.

When enabled, you can configure multiple authentication methods to secure your webhook endpoint:

  • Secret Token: A shared secret token that must be provided in webhook requests
  • HMAC Signature: Validates request payload integrity using HMAC-SHA256 signatures
  • IP Whitelisting: Restricts webhook access to specific IP addresses

Security Note: It is highly recommended to enable at least one authentication method for production environments to prevent unauthorized webhook requests.

If disabled, the webhook endpoint will accept requests from any source without authentication (not recommended for production).
