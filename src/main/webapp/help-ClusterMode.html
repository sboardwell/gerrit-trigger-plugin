Enable Cluster Mode for High Availability/High Scalability (HA/HS) deployments.

When enabled, this plugin uses an embedded Hazelcast cluster to coordinate event processing, build tracking, and job cancellation across multiple Jenkins replicas. Each Jenkins replica runs its own Hazelcast member that forms a peer-to-peer cluster with other replicas. This prevents duplicate builds and ensures consistent state across the cluster.

Note: Cluster mode is designed for CloudBees Core CI HA/HS deployments. For single-instance Jenkins installations, leave this option disabled.

Changing this value requires a Jenkins restart to take effect.
