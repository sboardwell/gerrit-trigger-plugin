A shared secret token used to authenticate webhook requests from Gerrit.

The token must be provided by Gerrit in one of the following ways:

  • In the X-Gerrit-Token HTTP header
  • In the Authorization: Bearer <token> HTTP header
  • As a query parameter: ?token=<token>

Configuration in Gerrit: You need to configure this same token in your Gerrit server's webhook configuration for the webhook to be accepted.

Note: The token is stored encrypted in Jenkins. Choose a strong, random value for production use.
