Events are received via HTTP POST webhooks from the Gerrit server, while maintaining SSH connection for sending build results.

How it works:

  • Gerrit sends events as HTTP POST requests to Jenkins webhook endpoint
  • Jenkins receives and processes events through the webhook endpoint
  • SSH connection is still maintained for sending build results back to Gerrit (review comments, votes)
  • Webhook authentication can be configured (secret token, HMAC signature, IP whitelisting)

Webhook Endpoint:

$JENKINS_URL/gerrit-webhook/

Important: The trailing slash (/) is mandatory. Ensure your Gerrit webhook configuration includes it.

Requirements:

  • Gerrit server must be configured with webhook plugin or built-in webhook support
  • HTTP/HTTPS connectivity from Gerrit to Jenkins must be available
  • SSH access is still required for sending build results back to Gerrit
  • Webhook authentication should be configured for production environments

Benefits:

  • Firewall-friendly: Uses standard HTTP/HTTPS protocols instead of persistent SSH connections
  • Flexible authentication: Supports multiple authentication methods (tokens, HMAC, IP filtering)
  • Scalable: HTTP POST is stateless and easier to load balance
  • No persistent connections: No need to maintain long-lived SSH connections for event delivery
  • Cloud-friendly: Works well with reverse proxies, load balancers, and cloud environments

Considerations:

  • Requires webhook configuration on the Gerrit server side
  • Still needs SSH connection for sending build results (bidirectional communication not possible over webhooks)
  • Should enable webhook authentication to prevent unauthorized requests
  • May have slightly higher latency than SSH streaming (depends on Gerrit webhook delivery)

Security Configuration:

When webhook authentication is enabled, you can configure:

  • Secret Token: Shared token validated from HTTP headers or query parameters
  • HMAC Signature: Cryptographic validation of payload integrity using HMAC-SHA256
  • IP Whitelisting: Restrict access to specific Gerrit server IP addresses
